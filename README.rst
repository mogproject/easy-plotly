===========
Easy Plotly
===========

Unofficial extension of Plotly for Python.

------------
Installation
------------

+-------------------------+---------------------------------------+
| Operation               | Command                               |
+=========================+=======================================+
| Install                 |``make install``                       |
+-------------------------+---------------------------------------+
| Uninstall               |``make uninstall``                     |
+-------------------------+---------------------------------------+
| Upgrade                 |``make upgrade``                       |
+-------------------------+---------------------------------------+
| Check installed version |``make version``                       |
+-------------------------+---------------------------------------+
| Launch Jupyter Lab      |``make lab``                           |
+-------------------------+---------------------------------------+
